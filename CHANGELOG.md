- Progressive Web App (PWA) support
- Service worker for offline functionality
- PWA manifest with app metadata
- App icons in multiple sizes (192x192, 512x512, SVG)
- Fancy icon design with gradient background, code brackets, and AI agent nodes
- OpenStreetMap tile caching for offline use
- Automated version bumping via GitHub Actions
- CHANGELOG.md for tracking versions

### Changed
- Updated icon from vite.svg to custom PlaneCode icon
- Enhanced meta tags for PWA support
